Basic usage

It's very easy to start with the tsFramework. Let's do it together.

How the framework works

A request to the framework will be treated like that:

  1. Client requests an URL for e.x. tsfw.localhost/home
  2. Server takes the request and guide it to the tsfw_src/framework.php, here begins the magic
  3. The framework looks up a route in the file settings/routes.json that matches the URI /home
  4. tsFramework invokes the controller given for the route and gets the response from it
  5. It sends the response to the client

That's already everything you have to know about it.

How to create a static page

Okay now we want to create our first static page accessable under the route /basic-usage. For that we do the following:

  1. Create a route entry

    Open the file site/settings/routes.json in your favourite editor.

    Insert at the end of the file the following code:

    "basic-usage": { "pattern": "/basic-usage", "controller": "ch:timesplinter:controller:StaticPageController:getPage", "method": "GET" }

    You have to separate the new entry with a comma (,) from the entry before. The StaticPageController:getPage does only look up in the dir site/temapltes/default/pages if a file with the key of the route exists, renders it and sends it as response.

  2. Create the HTML file

    Now we've to create the file in the path site/temapltes/default/pages and fill it with some content. For e.x. as follows:

    <h1>It works</h1> <p>hello world</p>
